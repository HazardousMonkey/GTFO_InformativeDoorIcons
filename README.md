## Informative Door Icons -- Client-side

<p align="center">This mod adds visual QoL to the various types of door icons. Color changes, sprite replacements, bug fixes, etc. Don't like what you're seeing? Toggle or change most features via the Gale/R2 config editors, even during gameplay.</p>
